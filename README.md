# HKInformationSystem

Java-based information management system demonstrating object-oriented design, search functionality, document retrieval, and unit testing using JUnit.

## Overview
Console-based application that allows users to log in, search for documents, and retrieve document details. The system is structured to demonstrate clean class design, modular logic, and testable functionality.

## Features
- User authentication (basic login validation)
- Search functionality with result handling
- Document retrieval using `viewDocument()`
- Exception handling for invalid inputs
- Unit testing using JUnit
- Observer pattern for notifications (if implemented)

## Testing
JUnit tests cover:
- Successful and failed login scenarios  
- Search with and without results  
- Valid and invalid document retrieval  
- Edge case handling

git clone https://github.com/nagarmanya07/HKInformationSystem.git

Open the project in IntelliJ and run:
- HkInformationSystem.java (main application)
- HkInformationSystemTest.java (unit tests)
